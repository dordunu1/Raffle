# YouTube Thumbnail Design Prompt

## Thumbnail Design Prompt for FHE Raffle Demo

**Style**: Modern, tech-forward, eye-catching thumbnail design

**Main Elements**:
- **Background**: Dark gradient (deep purple/blue to black) with subtle blockchain network pattern overlay
- **Central Focus**: Large, bold text "FHE RAFFLE" in bright orange (#FB923C) with glowing effect
- **Subtitle**: "Provably Fair" in white text below main title
- **Visual Elements**:
  - Lock icon with encryption symbols (representing FHE security)
  - Ethereum logo or blockchain network visualization
  - Prize pool visualization (coins/tokens floating)
  - "5 WINNERS" badge in corner
- **Color Scheme**: 
  - Primary: Orange (#FB923C) - matches app theme
  - Accent: White/light gray for text
  - Background: Dark purple/blue gradient
  - Highlights: Gold/yellow for prize elements

**Text Overlay**:
- Main Title: "FHE RAFFLE" (large, bold, orange)
- Subtitle: "Provably Fair Decentralized Raffles" (medium, white)
- Bottom Right: "LIVE DEMO" badge with red accent

**Mood**: Exciting, trustworthy, cutting-edge technology

**Technical Details**:
- Resolution: 1280x720 pixels (16:9 aspect ratio)
- High contrast for visibility on small screens
- Bold, readable fonts
- Minimal text (3-5 words max for readability)

## Canva / Design Tool Instructions

1. **Background**: Use dark gradient (purple #2D1B69 to black)
2. **Main Text**: "FHE RAFFLE" - Font: Bold Sans-serif, Size: 80-100pt, Color: #FB923C, Add glow/shadow effect
3. **Subtitle**: "Provably Fair" - Font: Medium Sans-serif, Size: 40pt, Color: White
4. **Icons**: Add lock/encryption icon, Ethereum logo (small, top corner)
5. **Badge**: "LIVE DEMO" - Red background, white text, rounded corners
6. **Accents**: Add subtle blockchain network pattern overlay at 20% opacity
